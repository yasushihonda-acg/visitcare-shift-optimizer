# ADR-003: 移動時間計算の実測値利用

## ステータス
承認済み

## コンテキスト
直線距離での計算では、川や線路、一方通行などを考慮できず、実現不可能なシフトになるリスクがある。

## 決定
Google Maps Distance Matrix API を使用する。

## 補足
APIコスト削減のため、取得した「地点間移動時間」はFirestoreにキャッシュし、再利用する設計とする。

## 現行システムとの関係
現行Excelの Location シートに 2,401 x 2,401 の移動時間マトリクスが存在。
値の大半が「5」（分単位と推定）→ Google Maps実測値への置き換えが必要。
